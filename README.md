# Métodos de clusterização (K-Means, DBScan e MeanShift)

Utilização de métodos de clusterização da biblioteca Scikit-Learn

O objetivo é entender o que é clusterização na prática. Para isso, no notebook começo com uma análise exploratória dos dados, para isso utilizo uma biblioteca chamada Biokit para gerar os gráficos, como o de correlação.

Depois disso, inicio os métodos de clusterização. Para não ter que implementar esses métodos do zero, utilizo a biblioteca do Scikit Learn, na parte específica de clustering (agrupamento).

Os métodos utilizados são: o <strong>K-Means</strong>, <strong>Mean-shift</strong> e além deles, o <strong>DBSCAN</strong>. Três métodos bastante utilizados para agrupar. Aqui utilizo uma base de dados de vinho.

Inicio com a análise exploratória para ver como os atributos se comportam, e além disso, para aplicar os algoritmos de clusterização. Então, avalio a clusterização, utilizando o método chamado coeficiente de silhueta.

Utilizo também a biblioteca <strong>Plotly</strong> para gerarmos os gráficos 3D.

Em resumo o resultado final desse projeto é fazer uma análise exploratória, gerar um gráfico de correlação, agrupar os dados utilizando o DBSCAN, o Mean-hift e o K-Means, que são os métodos mais utilizados de clusterização. Além disso, utilizo o coeficiente de silhueta e construir gráficos com o Plotly.
